Quick Start

The JTF News screen saver runs in any web browser. To use it as your system screen saver, you'll need a wrapper app for your platform.

URL Parameters

Customize behavior by adding parameters to the URL:

Parameter Default Description
audio=true false Enable text-to-speech audio for news
interval=30 50 Seconds between background image changes
gap=60 45 Seconds of silence between news stories
Example with audio enabled https://jtfnews.org/screensaver.html?audio=true

Self-Hosting

Want to run your own JTF News feed? You can point the screen saver to custom data sources:

Parameter Description
images=URL URL to custom image manifest JSON
stories=URL URL to custom stories feed JSON
Note: Custom URLs must allow CORS requests, or be on the same domain as the screen saver.
